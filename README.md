# Bookworm Adventures - Word Battle Game 📖⚔️

A Bookworm Adventures-inspired word battle game built in VB.NET where
you defeat enemies by forming words from a 4x4 letter grid.

---

## How to Play

- You are given a 4x4 grid of random letters
- Click letters to form a valid English word
- Submit the word to deal damage to the enemy
- Longer and rarer words deal more damage
- The enemy attacks back every turn
- Your HP and the enemy's HP are shown as bars on screen
- Defeat the enemy before your HP runs out!

## Requirements

- Windows PC
- Visual Studio 2019 or later (Community Edition is free)
  Download: https://visualstudio.microsoft.com/
- .NET Framework 4.7 or later (comes with Visual Studio)

---

## How to Clone and Run

1. Open a terminal or Git Bash
2. Run this command:

   git clone https://github.com/YOUR_USERNAME/BookwormAdventures.git

3. Open the folder and double-click the .sln file
4. Visual Studio will open the project
5. Make sure the dictionary file and music file are in the correct folder
6. Press F5 to run

---

## How to Download Without Git

1. Go to the GitHub repo page
2. Click the green "Code" button
3. Click "Download ZIP"
4. Extract the ZIP folder
5. Open the .sln file in Visual Studio
6. Press F5 to run

---

## Important Notes

- The dictionary .txt file must be in the same folder as the .exe
- The background music file must also be in the same folder as the .exe
- If the game crashes on launch, check that both files are present
